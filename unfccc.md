## UNFCCC dataset construction 

The final dataset includes documents covering the listed topics, excluding some (irrelevant) document types. 

### Included Topics

| Included Topics |  |
|-----------------|--|
| Action for Climate | Climate Finance |
| Empowerment | Consultative Group of Experts |
| Adaptation | COP Decisions and Agreements |
| Agriculture and Food Security | Global Climate Action |
| Ambition and Collective Progress | Global Stocktake |
| Biennial Reports | Just Transition |
| Biennial Update Reports | Local Communities and Indigenous People Platform |
| Capacity Building | Mitigation |


### Excluded Document Types

| Excluded Types |  |
|----------------|--|
| Biographies | Forms |
| Catering | Manuals |
| Brochures | List of Participants |
| Guidelines | Daily Programs |
